# pythmatics

Pythmatics adalah platform kuis matematika interaktif berbasis web yang dirancang untuk membantu pelajar memahami konsep kombinatorika secara terstruktur, cepat, dan aplikatif.

Project ini dibangun sebagai aplikasi statis berbasis HTML, CSS, dan JavaScript dengan fokus pada pengalaman pengguna yang sederhana dan responsif.
